# ELO图像评分系统

一个基于ELO算法的图像评分系统，通过两两比较的方式对图像进行评分排序。

## 功能特性

1. **图像导入**: 手动选择包含大量图像的目录，自动导入所有图像
2. **图像列表**: 左侧显示图像名称、目录和对应的ELO分数
3. **图像查看**: 双击图像名称可以查看大图，图片自适应窗口全屏显示
4. **参数设置**: ELO系统所有参数用中文显示，可以自定义并保存
5. **匹配界面**: 随机选择图像对让用户选择，实时计算ELO分数
6. **数据保存**: 图像名和对应分数可以保存为JSON文件
7. **数据加载**: 从JSON文件加载分数，包含格式检查

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
python elo_system.py
```

2. 点击"选择图像目录"按钮，选择包含图像的文件夹

3. 在参数设置区域调整ELO参数（K因子和初始评分）

4. 点击"进入匹配界面"开始进行图像比较

5. 在匹配界面中，选择你更喜欢的图像，系统会自动更新ELO分数

6. 点击"保存分数"将当前所有图像的分数保存到JSON文件

7. 点击"加载分数"从JSON文件恢复之前的分数

## ELO算法参数说明

- **K因子**: 控制每次比赛后分数变化的幅度，值越大变化越快（默认32）
- **初始评分**: 新加入图像的初始ELO分数（默认1500）

## 文件说明

- `elo_system.py`: 主程序文件
- `elo_config.json`: 保存ELO参数配置
- `elo_scores.json`: 保存图像分数数据
